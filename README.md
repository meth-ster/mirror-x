# mirror-x
It's half-assed, don't look at it. I'm just messing around with some Python stuff.

## What it does
mirror-x is a pretty basic script that mirrors some input. That's about it. It takes a string, flips it around, and prints it back out.

## Installation
You'll need Python 3.x to run this thing. Just clone the repo and you're good to go. You can install the required packages with pip:
```bash
pip install -r requirements.txt
```
Though, there aren't really any requirements besides Python.

## Running
To run mirror-x, just execute the script:
```bash
python mirror_x.py
```
You can pass a string as an argument if you want:
```bash
python mirror_x.py "hello"
```
## Example
If you run `python mirror_x.py "hello"`, it'll output `olleh`. Like I said, it's not super complicated.
